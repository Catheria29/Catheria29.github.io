## KNN
 **K-Nearest Neighbors (KNN)** does not really learn. And guess what ? it simply memorizes the entire training dataset. Let me explain with a simple visual tutorial.

What is KNN? KNN is a supervised machine learning algorithm used for classification and regression. It predicts the label of a new data point by finding the closest labeled examples in the training dataset and then selecting the majority class for classification or averaging values for regression.
Consider this small example training set. The data consists of x1 and x2 coordinates and a label:**(1,1) CAT, (2,2) CAT, (3,1) CAT, (1.5,4) CAT, (1,7) DOG, (2,6) DOG, (5,6) DOG, (6,2) FROG, (7,4) FROG, (8,6) FROG**. Visualizing these points, cats is blue, dogs green, and frogs purple.
Now, suppose we want to predict the label for a new point with coordinates (1,2). How does KNN work?
 It calculates the distance from this new point to every single point in the training set.
 
For a better understanding of how KNN works internally, consider the case where k equals 1. The algorithm selects the single closest point to the input and assigns that point's label. In this case, the closest point to (1,2) is (1,1) which is a cat. Therefore, with k = 1, the prediction is cat.
Now, consider the case where k equals **3**. The algorithm selects the three closest points to the input. Let us assume the three closest points are (1,1) which is cat, (2,2) which is cat, and (1.5,4) which is also cat. In this scenario, all three are cats, so the prediction remains cat. However, to illustrate how k = 1 and k = 3 can give different results, imagine a different new point, for example (2,5). With k = 1, the closest point might be (2,6) which is dog, so the prediction would be dog. But with k = 3, the three closest points could be (2,6) dog, (1,7) dog, and (5,6) frog. That gives two dogs and one frog, so the majority vote would be dog as well. For a true difference, consider a point like (3,5). With k = 1, the closest point might be (2,6) dog, so prediction dog. With k = 3, the three closest points could be (2,6) dog, (5,6) frog, and (3,1) cat. That gives one dog, one frog, and one cat. In case of a tie, a tie-breaking rule is applied, which could result in a different label than the k = 1 case. This shows that choosing different values of k can change the final prediction, especially when the point lies near decision boundaries between classes.

The key insight is that the model memorizes all the training points. For each prediction, if we have n points, it computes the distance from the input to all n points and then selects the k smallest distances. Imagine a dataset with one billion points. That would require one billion distance calculations for every single prediction, which is extremely slow and memory intensive. The main takeaway is that KNN is simple and intuitive model.
