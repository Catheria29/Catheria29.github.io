**Tech:** Python, Scikit-Learn, Streamlit
Developed a diagnostic tool achieving **99.6% Accuracy** to predict pregnancy complications. Engineered a clinical dataset and validated logic via Confusion Matrices.

**Tech:** Java, Evolutionary Computing
Solved NP-hard constraint problems using **Genetic Algorithms**. Designed custom fitness functions and mutation logic to optimize search in high-dimensional spaces.
